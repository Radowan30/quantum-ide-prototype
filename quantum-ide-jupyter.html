Grover's Algorithm Implementation

circle Kernel Idle

This notebook demonstrates a basic implementation of Grover's algorithm for unstructured search. We will construct a quantum circuit to find a marked item in a small database.

[1]:

Code Cell 1: Import Libraries

from qiskit import QuantumCircuit, Aer, execute
from qiskit.visualization import plot_histogram
import numpy as np
[2]:

Code Cell 2: Define Quantum Circuit

# Define the number of qubits
n = 2
qc = QuantumCircuit(n)
# Apply Hadamard gates to all qubits
for qubit in range(n):
    qc.h(qubit)
# Oracle for marked state |11>
qc.cz(0, 1)
# Display the circuit
qc.draw('mpl')

[3]:

Code Cell 3: Run Simulation and Get Results

# Measure the qubits
qc.measure_all()
# Execute the circuit
simulator = Aer.get_backend('qasm_simulator')
counts = execute(qc, simulator, shots=1024).result().get_counts()
# Plot the results
plot_histogram(counts)
